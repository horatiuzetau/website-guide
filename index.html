HTML 5
  • tags (div, p, h1, h2, etc..)
  • ids (unique identifier for element)
  • classes (not unique identifier - reusable)
  • attributes (additional properties of elements)
  • DOM (Document Object Model)
CSS (Sass)
  • selectors (classes, ids, elements, etc..)
  • pseudo-elements (::before, ::after, ::first-line, etc..)
  • pseudo-classes (:hover, :focus, etc..)
  • media queries (used for responsiveness)
  • mixins (used for reusable style blocks)
  • variables (Sass related)
JavaScript
  • selectors (getElementById, getElementByClassName)
  • event listeners (addEventListener method)
  • AJAX (Async Javascript and XML)
Adobe XD
  • basic shapes
  • pen tool
  • critical properties are easy to modify
  • layering system
  • delivering fast and easy to create designs
